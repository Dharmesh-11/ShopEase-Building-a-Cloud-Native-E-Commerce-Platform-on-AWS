# ☁️ AWS S3 Setup – ShopEase Project (Task 3)

## 📌 Overview
In this task, Amazon S3 (Simple Storage Service) is used to:

- Host a static website
- Store application assets (CSS, JS, Images)
- Manage logs with lifecycle rules
- Store database backups

---

## 🎯 Objectives

- Create S3 buckets
- Enable static website hosting
- Configure public access
- Apply bucket policy
- Enable versioning
- Create lifecycle rules

---

# 📜 6. Bucket Policy

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadAccess",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::shopease-static-assets-dharmesh/*"
    }
  ]
}
```
